ScummVM

ScummVM isn't an emulator - it's a full reimplementation of numerous game engines. It allows hundreds of classic games to be played on a range of devices they were never meant to support, from modern PCs to the PS Vita and WII.


ScummVM Homepage

A1: Conceptual Architecture

This report is a breakdown of the conceptual architecture of ScummVM and the SCI engine. It also covers the evolution of the software over time, data and control flow between components, and the distribution of developer responsibilities.


Report PDF Presentation Slides

A2: Concrete Architecture

In this report we analyze the concrete architecture of ScummVM and the SCI engine, in comparison with the conceptual architecture defined in A1. We discuss the differences between these architectures and their rational, as well as how the concrete architecture was derived from the project's source code and documentation.
